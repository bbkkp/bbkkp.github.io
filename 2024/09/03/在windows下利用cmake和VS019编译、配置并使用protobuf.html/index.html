2024-09-03
开源库-protobuf

下载protobuf

下载地址https://github.com/protocolbuffers/protobuf/releases
注意请不要下载高版本，大于22的版本的CMakeLists.txt文件在根目录，最关键的是第三方文件夹third-party下的第三方库都是空的，我们需要自己clone,而且在之后的配置过程中还会遇到一些链接问题又或者其他因为版本不同导致的奇怪问题，我建议不要这样。

解决方法

下载小于22的版本，它的版本号应该形如3.21.xx,不要在意多了个3,可能是google改了版本策略。

重点配置

接下来的操作少一步都能烦死你

  1. 属性页VC++包含目录include
  2. 属性页VC++库目录bin
  3. 属性页C/C++代码生成运行库多线程调试 DLL (/MDd)
  4. 链接输入附加依赖项libprotobufd.lib,libprotocd.lib
  5. pb.h文件添加宏定义 #define PROTOBUF_USE_DLLS
  6. 确保你的protobuf版本和你生成pb文件的版本保持一致，有时候你以为的环境变量不是你以为的，在软件逐渐增多之后，某个大型软件下是可能存在protoc.exe,到时候会出现麻烦无比的版本问题，而且你需要花费很长时间才能排查这里。
