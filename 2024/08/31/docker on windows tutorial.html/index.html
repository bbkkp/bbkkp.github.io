docker在windows上的使用教程

2024-08-31
工具-docker

目标

我们会介绍docker的基本使用并最终通过Windows上的vscode连接到本地的docker容器中进行开发，你会体验到docker相比于虚拟机无语伦比的强大。[当然如果你极致追求隔离性和安全性，或者某些特殊的系统，虚拟机还是首选]

拉取镜像

我们以经典的的ubuntu18.04为例
docker pull ubuntu:18.04
这里会默认从官方仓库docker.io/library/ubuntu中拉取标签为 18.04 的镜像

创建对应镜像的容器

docker create -it ubuntu:18.04

查看存在容器

docker container ls --all

启动容器

docker start id/容器名

查看启动容器

docker ps

进入容器

docker run -it --rm ubuntu:18.04 bash

退出容器

exit

运行容器

docker run -it id /bin/bash

查看镜像

docker image ls

参数

  • it：这是两个参数，一个是 -i：交互式操作，一个是 -t 终端。我们这里打算进入 bash 执行一些命令并查看返回结果，因此我们需要交互式终端。
  • rm：这个参数是说容器退出后随之将其删除。默认情况下，为了排障需求，退出的容器并不会立即删除，除非手动 docker rm。我们这里只是随便执行个命令，看看结果，不需要排障和保留结果，因此使用 –rm 可以避免浪费空间。
