数据库基本指令

2024-08-31
工具-mysql

显示数据库

show databases;

创建数据库

create database databasename;

删除数据库

drop database databasename;

选择数据库

use databasename;

跳转数据库

use databasename;
先在当前数据库中显示数据库表，然后选择数据库表跳转过去。但是在当前数据库下就只能访问当前数据库的表

显示数据库中表

show tables;

显示表的信息

show clumns from tablename;

删除表

drop table tablename;
